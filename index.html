FAQ

Is it safe?
Yes. The app uses modern encryption standards and does not store any personal data. All traffic is securely tunneled and protected from third-party access.
Can I use my own server?
Of course. HitWave is compatible with custom servers. You just need to generate a link with the correct configuration and paste it into the app to connect.
Where can I get a key?
You can get a configuration key from any VPN provider that supports the XRAY-CORE or WireGuard protocols. Alternatively, you can set up your own server and generate the key yourself.
